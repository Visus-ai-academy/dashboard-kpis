Dashboard KPIs is a greenfield SaaS project for managing commercial goals, configurable KPIs, and gamification/endomarketing with visual dashboards and individual data entry by sellers.

**Why:** Visus is building a product for commercial teams to track goals, rank sellers, and gamify performance.

**How to apply:** All work must consider multi-tenant isolation (companyId in every query), the organizational hierarchy (Company > Units > Sectors > Teams > Sellers), and the dual auth model (NextAuth for admins, UUID token for sellers). Phase 1 (Foundation) must be completed before any other phase begins.
